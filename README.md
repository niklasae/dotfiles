dotfiles
========

## Installation ##

Clone the repo:

    git clone ssh://git@github.com/niklasae/dotfiles.git ~/dotfiles

The run prereq and setup

    ./prereq.sh
    ./setup.sh

They will install dependencies, setup symlinks and create relevant directories.
